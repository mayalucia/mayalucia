* Conventions

- One directory per experiment: =experiments/NN-short-name/=
- Each directory contains =notebook.org= (the primary record) plus data, scripts, outputs
- Status: =seed= → =active= → =paused= → =harvested= → =composted=
  - *seed*: question posed, not yet explored
  - *active*: currently being worked on
  - *paused*: parked, may resume
  - *harvested*: key insights extracted, usable elsewhere
  - *composted*: dead end, but decomposed into lessons
- Each experiment links back to relevant domain docs (parbati.org, mayalucia.org, etc.)
- Code blocks are for reproducibility; the prose is the primary record
- Tag experiments with domains: =parbati=, =brain=, =method=, =tool=
