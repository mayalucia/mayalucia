** Key insight

The /summoned vs companion/ distinction emerged as the deepest
structural gap. gptel-agent is summoned — the human fires a query,
the agent responds, the session pauses. Claude Code is companion — it
runs continuously, perceiving events, holding thread, depositing warps.
Powers that require continuous perception (attend-working-context,
hold-the-thread, lay-the-warp) cannot operate in a summoned harness
without an event bridge.
