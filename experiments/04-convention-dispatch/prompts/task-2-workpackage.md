# Task: Work Package for Story Browser Geological Cross-Section

The story browser (`website/story-browser/`) displays stories on a
map. We want to add a geological cross-section view: a vertical
transect through the Parbati valley showing rock layers, fault lines,
and hot spring locations, with story markers positioned at their
geographic coordinates projected onto the cross-section.

Draft a work package for this feature. The implementation will use
ClojureScript and Reagent (the existing story browser stack). The
cross-section data can come from published geological surveys of
the Kullu-Manikaran fault system.

Write the work package as an Org-mode document.
