What This Is

The Physics

A quantum sensor is a dialogue between a quantum system and the world, mediated by light, shaped by feedback. The magnetometer does not find the magnetic field — it participates in a dance between the atomic spin and the external field, and the number that comes out is not the field itself but what we can say about the field given our apparatus, our noise floor, our model.

The Cramer-Rao bound sets the speed limit on this knowledge. Fisher information — the curvature of the likelihood — determines how distinguishable two nearby field values are. No unbiased estimator can do better. This is not a limitation. It is the grammar of measurement.

Bloch Equations — The Shared Core

A spin-1/2 particle in a magnetic field B precesses at the Larmor frequency omega_L = gamma |B|. The state is captured by the Bloch vector M = (M_x, M_y, M_z), living on or inside the unit sphere. The equations of motion:

dM/dt = gamma (M x B) − (M_x/T_2,  M_y/T_2,  (M_z − M_0)/T_1)

Three terms, three physics: precession (the spin rotates about the field — pure, reversible, unitary), transverse relaxation (T_2 — decoherence, loss of phase information), and longitudinal relaxation (T_1 — thermalisation toward equilibrium M_0).

These equations govern atomic magnetometers and NMR and MRI. The digital twin of a magnetometer and the digital twin of an MRI pulse sequence share this mathematical core.

Applications

We carry electromagnetic fields in our heads. Every cortical column produces a femtotesla whisper — drowned by the Earth's field, by power lines, by thermal noise — yet it carries information. The same sensors built for quantum optics are now imaging the living brain.

Wearable MEG

Optically pumped magnetometers — the same alkali-vapor technology as laboratory atomic magnetometers — are replacing cryogenic SQUIDs for brain imaging. Room temperature. Millimetres from the scalp. Subjects can move naturally.

Single-Neuron Magnetometry

NV-diamond sensors at nanometre scales can detect the magnetic field of individual action potentials. Proof-of-concept demonstrated with worm ganglia. The physics differs from alkali vapors; the controller grammar does not.

Ultra-Low-Field MRI

Portable brain imaging at microtesla fields, using quantum magnetometer arrays instead of superconducting magnets. The same Bloch equations. The same digital twin, with a structured sample.

Magnetocardiography

The heart's magnetic field — a thousand times stronger than the brain's — measured non-invasively, without electrodes, through clothing. Portable MCG for point-of-care cardiac screening.

Point-of-Care Diagnostics

Chip-scale NMR for blood chemistry. Portable MEG for bedside stroke detection. The bottleneck is not the sensors — it is the control and calibration. Exactly what a universal controller addresses.

Quantum Optics Lab

Where it begins: atomic magnetometers, gravimeters, optical clocks. Each controller built from scratch, each time. The grammar that emerges from building digital twins here propagates to every other domain.

The connections are structural, not analogical. The same Fisher information bounds the magnetometer's sensitivity and the MEG system's source localisation. The same Kalman filter tracks a drifting field and a neural ensemble. The same noise models apply whether the source is a rubidium cell or a human brain.

Architecture

MāyāPramāṇa implements a functional programming grammar for sensor control: pure core for physics, effectful shell for hardware, monadic composition for the pipeline from photon to inference.

Language Role Strength
Python Interactive exploration, prototyping, plots Fast iteration, org-babel integration
Haskell Executable specification, QuickCheck properties Types as physics, referential transparency
C++ Deployment, performance, FPGA-adjacent Zero-cost abstractions, real-time guarantees

All three must agree on the same physics. Cross-language validation follows the pattern: Python generates reference data, Haskell and C++ load and compare. If they disagree, the physics is wrong somewhere — and the tests will tell you where.

Lessons

Each lesson is a literate org document that builds understanding from first principles: physics → Python exploration → Haskell specification → C++ deployment → cross-language verification → exercises. The code is tangled from the document; the document is the source of truth.

00 — Bloch Equations: Spin Dynamics as Pure Functions

The Bloch equations implemented in all three languages. The same physics, three expressions. The RK4 integrator, the relaxation dynamics, the equilibrium recovery — each language reveals different aspects of the structure.

Python @dataclass class BlochState: mx: float my: float mz: float def bloch_rhs(state, params, B): """dM/dt: precession + relaxation""" cross = gamma * np.cross(M, B) relax = [-mx/t2, -my/t2, -(mz - m0)/t1] return cross + relax
Haskell data BlochState = BlochState { mx :: Double , my :: Double , mz :: Double } blochRHS :: BlochParams -> Vec3 -> BlochState -> BlochState blochRHS p b s = precession p b s `add` relaxation p s
C++ struct BlochState { double mx, my, mz; }; auto bloch_rhs( const BlochState& s, const BlochParams& p, const Vec3& B) -> BlochState { return precession(s,p,B) + relaxation(s,p); }
