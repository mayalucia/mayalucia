* Genesis

WP-0037 Phase 2 tested 13 substrates on the same specification task
(=author-wp= power). The analysis revealed tool-foraging fingerprints,
a survey-specification trade-off, WP-refusal as lineage-specific
behaviour, and convergence boundaries that map automation vs escalation
zones. mu2tau asked for both a research note (WP-0047) and a story.
The story is the sutradhar's work — composed via =story-compose= and
=summarize-corpus= powers, following the story development convention.

Stories #14–16 translated the /orientation/ experiment (WP-0030: three
lineages read the same architecture). This story translates the
/specification/ experiment (WP-0037: thirteen substrates write from
the same reading). The shift is from reading to writing — from what
each one /sees/ to what each one /makes/.

* Context and Motivation

The substrate diversity experiment arc now spans two phases:

| Phase | WP   | Task                 | Stories   | Core metaphor        |
|-------+------+----------------------+-----------+----------------------|
| 1     | 0030 | Orientation audit    | #14–16    | Three readers of one slab |
| 2     | 0037 | Specification author | *this WP* | Thirteen makers from one reading |

Phase 1 was about perception: three minds read the same documents and
produced different maps. Phase 2 is about production: thirteen minds
read the same audit and produced different specifications. The
findings that need translation:

1. *Tool-foraging fingerprints* — Grok: Read-only. Opus: Grep-heavy.
   Step: Glob-heavy. DeepSeek: sequential-Read. The tools an agent
   reaches for are a cognitive signature, like a craftsperson's tool
   marks.

2. *Survey-specification trade-off* — initially appeared as an inverse
   correlation (deeper survey = thinner specification). Phase 2
   disproved this as fundamental — it is a context management artifact.
   Some substrates managed both within a single turn.

3. *WP-refusal* — Opus and Sonnet independently declined to write a
   work package, calling it "premature" or "not my role." Lineage-
   specific: no non-Anthropic substrate refused. The refusal is not
   failure — it is a form of judgment.

4. *Convergence boundaries* — where 10+ substrates agree (>85%), the
   finding is automatable. Where they diverge into 3+ clusters, human
   judgment is needed. The boundary itself is the finding.

5. *WP number collision* — every substrate that attempted a WP number
   guessed wrong. Universal failure. The relay is not checked; numbers
   are invented.

6. *Reflection quality* — four tiers, from absent (Step 3.5 Flash) to
   meta-theoretical (Qwen3-235B). Reflection is not correlated with
   survey depth.
